# 极简电视

极简电视直播App，简洁易用的iPTV播放器，支持Android、iOS欢迎下载体验！

#### 功能

- 简易的IPTV直播源播放器
- 可自主添加直播源链接
- 干净简洁的UI交互
- 自带国内直播源
- 增加对.m3u文件的解析支持
- 增加对.txt文件的解析支持(v1.5.0新增)
- 纯黑主题(v1.5.0新增)
